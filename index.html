What i learnt

 I learnt the format which is:

<!DOCTYPE>
That stands for document type and for this case it obviously is just html website.
<html lang;>
That is the open tag for html and must also indicates the language that the code is in using lang.
<head>
This is the part of the code that is not visible on the actual website but contains: Meta data that is picked up and used by the browser.Such metadata includes:
  1. The charset which iss the character set and is normally equated to UFT-8 for most websites
  2. The author which is basically the writter of the code and precides name and preceded by content where the actual name is filled in.
  3. The decription which like the author precideds name and is precided by content whereby in this case a literal short decription of the page and its contents are shared to keep the browser in the know.
